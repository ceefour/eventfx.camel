Advanced Event Notification Framework with Apache Camel
=======================================================

For more info, see:
http://spring-java-ee.blogspot.com/2010/01/advanced-event-notification-framework.html


Run from Eclipse
----------------

You'll need (recommended) SpringSource Tool Suite or Eclipse Java with
M2Eclipse.

Inside Eclipse IDE, run the project as Java Application like usual.


Run as Camel route
------------------

To run this router either embed the jar inside Spring
or to run the route from within maven try

    mvn camel:run
